# kr-culture-events-crawler

한국 지자체 문화행사 공지를 수집해 SQLite로 저장하고 요약 리포트를 출력하는 Python 프로젝트이다.

## 환경
- Python 3.11 이상

## 설치

Windows:
```bash
py -3.11 -m venv .venv
.venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

macOS / Linux:

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requiremetns.txt
```
